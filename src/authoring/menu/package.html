Contains the classes pertaining to the Workspace's MenuBar. It contains one main, abstract class: WorkspaceMenu, from which EditMenu, GameMenu, HelpMenu, ServerMenu, SettingsMenu.

The two main classes in this package are WorkspaceMenuBar, which houses the actual MenuBar, and WorkspaceMenu, the superclass for all of the Workspace's Menus. The other classes in the package are subclasses of WorkspaceMenu.
