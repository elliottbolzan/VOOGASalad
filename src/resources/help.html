Combination Behavior
Click Select the clicked entity. A selected entity can be deleted or resized.
Shift + Click Add the clicked entity to a group selection.
Drag & Drop Move the entity to a new position on the canvas.
Delete Key Delete the selected entity or entities.
Control Key + Click Add the entity selected in the panel to the canvas.
Control Key + Shift Key + Drag Add several copies of the entity selected in the panel to the canvas.
